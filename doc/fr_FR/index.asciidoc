== Google Agenda

=== Description
Le plugin agenda permet de récupérer les événements de son Google Agenda. Il gère les multiples évènements en les séparant par une virgule (“,”) il ne reste plus qu’à tester la présence de l’évènement dans un scénario pour déclencher une action (il existe dans les scénarios l’opérateur ~ pour savoir si une chaîne de caractères en contient une autre)
